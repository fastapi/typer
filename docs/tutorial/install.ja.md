# **Typer** をインストールする

最初のステップは **Typer** をインストールすることです。

まず [virtual environment](../virtual-environments.md){.internal-link target=_blank} を作成して有効化し、そのうえで次のようにインストールしてください。

<div class="termy">

```console
$ pip install typer
---> 100%
Successfully installed typer click shellingham rich
```

</div>

デフォルトでは、`typer` には `rich` と `shellingham` が含まれています。
